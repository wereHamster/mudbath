Continuous Deployment meets GitHub

Send hubot, deploy app to production to your chat,
Mudbath handles the rest.

Mudbath is a self-hosted continuous deployment server which is fully integrated into GitHub. The only way how it can be triggered is through the GitHub deployments API. That encourages you to simplify your deployment process and manage it through ChatOps tools.

The installation is simple, and there is very little to configure. This makes mudbath easy to get started with.

If you still have questions, suggestions, or just want to say hi, you can contact the author through email. If you discover bugs you can open issues in the github repository.
